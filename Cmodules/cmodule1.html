Getting Start with C

Hey there, before we begin to write programs in C, let’s find out the some information about see how it come into existence and how does it compare with other programming languages. C is a programming language developed at AT and T’s bell laboratories of USA in 1972 by Dennis Ritchie. It become popular cause it is simple and easy to use. Major part of popular operating system like windows, Unix, Linux and Android are written in C. Moreover, if one is to extend the operating system to work with new devices, one need to write device driver programs. These programs are written exclusively in C.

Constants, Variables and Keywords

The alphabet, digits and special symbols when properly combined for constants, variables and keywords. A constant is an entity that doesn’t change, whereas, a variable is an entity that may change. A keyboard is a word that carries special meaning. In programming languages, constant are often called literals whereas, variables are called identifiers. Let us now see what different types of constant and variable exist in C.

Types of Constants

Constants can be divided into two major categories
1. primary constant
2. secondary constant

Rules for constructing an integer constant
• An integer constant must have at least one digit.
• It must not have a decimal point.
• It can be any of zero positive or negative.
• No commas or blanks are allowed within an integer constant.
• The allowable range for integer constant is minus 2147483648 to +2147483647.

Rules for constructing real constants
• Hey real constant must have at least one digit.
• It must have a decimal point.
• It can be either positive or negative.
• No commas or blanks are allowed within a real constant.

Rules for constructing character constants
• A character constant is a single alphabet, a single digit or a single special symbol enclosed within single inverted commas.
• Both the inverted commas should point to the left.

Types of variables

A particular type of variable can hold only the same type of constant. For example an integer variable can hold only an integer constant, a real variable can hold only a real constant and a character variable can hold only a character constant.
Variables are the name of memory cells so bet they can store some type of value that can be changed further that’s why it’s called variable because its value can be vary.

Rules for constructing variable names
• A variable name is any combination of 1 to 31 alphabet’s, digits or underscores. Some compilers allow variable names whose length could be up to 247 characters. Do not create unnecessarily long variable name as it add to your typing effort.
• The first character in the variable name must be an alphabet or underscore( _ ). The variable name can't be start with a digit.
• A variable name don’t have any special symbol in it.

C keywords

Keywords are the words whose meaning has already been explained to the C compiler. There are only 32 keywords every level in C. The keywords should not be used as a variable names.

Form of a C program

There are certain rules about the form of C program that are applicable to all C programs.
• Each instruction in a C program is written as a separate statement.
• The statement in a program must appear In the same order in which we wish them to be executed.
• All statements should be in lowercase letters.
• C statement can be written anywhere in a given line. That’s why it is often called a free form language.
• Every statement must end with a semicolon (;).

Comments in C program

Comments are used in a C programme to clarify either the purpose of the program or the purpose of the some statement in the program. It is a good purpose to beginning program with a comment indicating the purpose of the program.
• Comments can be in small case, capital or a Combination. They should be enclosed within /* */ .
Example, /* First C program */

• ANSI C permits Comment to be written in another way as follows
// it is a comment
But it is for a single line comment only.

What is main() ?

main() forms a crucial part of any C program.
main() is a function. It is a container for a set of statements. A C program May have multiple functions. If it contains only one function its name has to main(). All statements that belong to main() are enclosed within a pair of braces {}.

Like functions in a calculator, functions in C also return a value. main() function always return in int value, hence is there an int before main() it is known as return type of the function. The integer value that we are returning is 0. 0 indicates success. If statements in main() fail to do their intend work, we can return a non-zero number from here this would indicate failure.

printf() and its purpose

C does not contain any keyword to display output on the screen. All output to screen is achieved using readymade library functions like printf(). • To be able to use the printf() function it is necessary to use #include "studio.h" at the beginning of the program. #include is a preprocessor directive its purpose will be clarified in further chapters.
• The general form of printf() function is,
printf(“format string”, list of variable);
Format string can contain,
%f for printing real value,
%d for printing integer values,
%c for printing character values
